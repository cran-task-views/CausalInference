Causal inference is a specific field of statistical analysis that looks into questions about effects of a treatment, intervention or policy on some target variable. Under certain model assumptions, causal inferences can be carried out by fitting simple regression models or combining several regression models in a specific way as will be sketched out later.

There do not exist basic R functions to do causal analyses straight away, but many methods - more or less complex - are implemented in different packages on CRAN, which we structure into main topics:

  • Methods for experimental data
  • Average treatment effect estimation
  • Heterogeneous treatment effect estimation
  • Policy learning
  • Causal discovery
  • Specific types of data
  • Specific application fields

If you think that we missed some important packages in this list, please contact the maintainer.

Specific types of data

  • Longitudinal data / time series and censored data : Imputation for time series is implemented in grf and grf. Other packages, such as grf, grf, grf, grf, grf, grf or grf, are dedicated to time series but also contain some (often basic) methods to handle missing data (see also MissingData). To help fill down missing values for time series, the grf and grf packages provides methods for imputing implicit missing values. Imputation of time series based on Dynamic Time Warping is implemented in grf for univariate time series and in grf for multivariate ones. grf also imputed data below the range for exploratory graphical analysis with the function impute_below. grf implements an estimation of the autoregressive threshold models with Gaussian noise and of positive-valued time series with a Bayesian approach in the presence of missing data. grf implements a probability weighted generalized estimating equations method for longitudinal data with missing observations and measurement error in covariates based on SIMEX. grf performs imputation for censored responses for interval data. grf proposes tools to benchmark missing data imputation in univariate time series.
  • Spatial data : Imputation for spatial data is implemented in grf using interpolation with spatial distance weights or kriging. grf is dedicated to satellite data and geostatistical interpolation of data with irregular spatial support is implemented in grf
  • Spatio-temporal data : Imputation for spatio-temporal data is implemented in the package grf using different methods as knn and SVD. Similarly, grf imputes missing values in daily precipitation time series accross different locations and grf imputes missing data air polluant concentrations.
  • Graphs/networks : Imputation for graphs/networks is implemented in the package dils to impute missing edges. grf provides a framework for analyzing Probabilistic Suffix Trees, including functions for learning and optimizing VLMC (variable length Markov chains) models from sets of individual sequences possibly containing missing values.
  • .. for contingency table is implemented in grf that can also be used for the analysis of contingency tables with missing data.
  • .. for compositional data (CODA) is implemented in grf (based on kNN or EM approaches) and in grf (various imputation methods for zeros, left-censored and missing data).
  • experiment handles missing values in experimental design such as randomized experiments with missing covariate and outcome data, matched-pairs design with missing outcome.
